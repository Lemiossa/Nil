# Nil

Nil is a simple 16-bit operating system on x86 for educational purposes.

*Attention: THIS IS NOT MY FIRST TIME TRYING TO DO AN OS, I MAY END UP DOING SOME PARTS MUCH FASTER THAN NORMAL*

## Third-party code

This project includes a small subset of the Bruce's C Compiler runtime
library (libc long arithmetic routines), required for 32-bit operations.
